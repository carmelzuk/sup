Basic

Add a bin dir to your path:

 

mkdir ~/bin

Add this to ~/.bashrc file:

PATH=$PATH:~/bin; export PATH

Composer

See composer download page

To install composer you need to have curl and php

 

sudo apt-get install curl

You also need the php json extension

sudo apt-get install php5-json

sudo curl -sS https://getcomposer.org/installer | php

 

Make the composer command easy to use:

(see "Basic" section for adding bin to path)

touch ~/bin/composer

vim ~/bin/composer

Write this to the file:

#!/bin/bash
php ~/tmp/composer.phar $@

and save. Make the script executable

chmod +x ~/bin/composer
